# Portafolio Profesional - Miguel Ángel Conde

## Descripción

Este proyecto corresponde a un **Portafolio Web Profesional** desarrollado con **React**, **Vite** y **Tailwind CSS** como parte del proceso de formación en **Desarrollo de Software**.

El objetivo del proyecto es presentar de forma organizada mi perfil profesional, habilidades técnicas, proyectos desarrollados y un formulario de contacto mediante una interfaz moderna, responsiva y de fácil navegación.

---

# Características

* Página principal (Hero).
* Sección **Sobre mí**.
* Sección de habilidades generadas dinámicamente desde un arreglo de datos.
* Sección de proyectos cargados mediante **Fetch API** desde un archivo JSON.
* Tarjetas reutilizables para habilidades y proyectos.
* Filtro de proyectos.
* Formulario de contacto con validación de datos.
* Footer con información de contacto.
* Navegación mediante anclas con desplazamiento suave (Smooth Scroll).
* Diseño moderno desarrollado con Tailwind CSS.

# Requisitos previos

Antes de ejecutar el proyecto asegúrese de tener instalado:

* Node.js (versión 18 o superior)
* npm (incluido con Node.js)
* Git

Puede verificar la instalación con los siguientes comandos:

```bash
node -v
npm -v
git --version
```

---

# Instalación

## 1. Clonar el repositorio

```bash
git clone https://github.com/SirMiguel15/portafolio_miguel.git
```

## 2. Ingresar a la carpeta del proyecto

```bash
cd portafolio_miguel
```

## 3. Instalar todas las dependencias

```bash
npm install
```

Este comando descargará automáticamente todas las dependencias definidas en el archivo **package.json**, incluyendo React, Vite, Tailwind CSS, React Icons y las demás librerías utilizadas por el proyecto.

## 4. Ejecutar el servidor de desarrollo

```bash
npm run dev
```

Al ejecutar el comando anterior aparecerá una salida similar a la siguiente:

```text
VITE v7.x.x ready

➜ Local: http://localhost:5173/
```

## 5. Abrir el proyecto

Abrir el navegador y acceder a:

```text
http://localhost:5173/
```
